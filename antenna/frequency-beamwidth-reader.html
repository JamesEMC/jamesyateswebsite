Graph reader for antenna datasheets

Read frequency vs half-power beamwidth plots into a table

Load a datasheet graph, set the rectangular plot area, click along one or two beamwidth curves, then export the raw points or an interpolated frequency table.

1. Image

Use a rectangular frequency versus beamwidth plot from a datasheet. Paste from clipboard also works.

2. Axis values

These are the values at the graph edges, not the sampled test points.

3. Sampling

Leave start and end blank to use the first and last selected points from the active trace data.

4. Click tools

Set bottom-left and top-right graph corners first. Then click along each curve from left to right.

Quick guide

  • Bottom-left is the graph corner at X min and Y min.
  • Top-right is the graph corner at X max and Y max.
  • Raw points are exactly what you clicked.
  • Sampled table interpolates between clicked points at the chosen frequency step. Blank start/end values use the selected point range.
